프로세스는 부모 프로세스의 자식으로써 부모 프로세스를 복사하여 생성되며 자식 프로세스는 부모의 코드 영역, 데이터 영역, 스택 영역, 그리고 PCB 내용을 모두 복사한다. 

- 자식 프로세스는 exit 함수로 부모 프로세스에게 종료를 알린다.
- 부모 프로세스는 자식 프로세스의 exit status를 읽고 자식 프로세스르 정리한다.
  - 자식 프로세스가 exit 신호를 보내지 못한 채 비정상적으로 종료되거나 
  - 부모 프로세스가 자식 프로세스보다 먼저 종료되어 exit status를 읽지 못하면
  - 해당 프로세스는 계속 메모리에 남게 됨
  - 이를 좀비 프로세스라고 부름
  - 컴퓨터를 오래 켜두면 느려지는 현상의 원인 중 하나
