# 핵심 포인트
- ✓ 세트 간 휴식 
  - 큰 근육 운동: 2-3분
  - 작은 근육 운동: 1-2분


- ✓ 점진적 과부하 
  - 매주 중량 또는 반복 횟수 조금씩 증가


- ✓ 영양 
  - 단백질: 체중 1kg당 1.6-2.2g
- ✓ 회복 
  - 평일 충분한 수면과 휴식
