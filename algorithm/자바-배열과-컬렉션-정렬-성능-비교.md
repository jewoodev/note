# Java

|                    | 정렬 방식                        | 시간복잡도                     |
|--------------------|------------------------------|---------------------------|
| Arrays.sort()      | DualPivotQuicksort           | 평균: O(nlogn) / 최악: O(n^2) |
| Collections.sort() | Timesort(삽입정렬과 합병정렬을 결합한 방식) | 최악: O(nlogn)              |

---

# 번외) 시간복잡도 비교
- 투 포인터: O(n)
- 정렬: O(nlogn)
