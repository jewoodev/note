# 롤링 배포와 블루그린 배포 비교

|        | 롤링 배포            | 블루그린 배포             |
|--------|------------------|---------------------|
| 배포 방식  | 하나씩 순차적으로 진행     | 동일한 서버군 생성 후 한방에 전환 |
| 배포 속도  | 느림               | 빠름                  |
| 문제 발생시 | 배포된 서버만 영향       | 모든 사용자가 영향, 롤백은 간단  |
| 비용     | 추가적인 서버가 필요하지 않음 | 일시적으로 두배의 리소스 필요    |
| 복잡성    | 상대적으로 단순         | 상대적으로 복잡            |
