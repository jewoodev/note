다이내믹 라우팅은 다시 여러가지 프로토콜로 나뉘어진다. 인터넷 상에 라우터는 셀 수 없을 만큼 많이 존재한다. 여기서 하나의 큰 조직이 많은 라우터를 연결해 만든 네트워크를 Autonomous System, **AS** 라고 부른다. 우리나라 AS 정보는 한국인터넷진흥원에서 확인할 수 있다.

AS가 있을 때 AS 내부에서 쓰이는 프로토콜을 인테리어 라우팅 프로토콜(Interior Routing Protocol, **IRP**)이라고 부른다. 그리고 서로 다른 AS를 연결하기 위한 라우팅 프로토콜을 Exterior Routing Protocol, **ERP**라고 부른다.

# Interior Routing Protocol
- Distance Vector
  - RIP, Routing Information Protocol
- Link State
  - OSPF, Open Shortest Path First

# Exterior Routing Protocol
- Path Vector
  - BGP, Border Gateway Protocol
