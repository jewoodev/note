# 비교 연산자와 동등성, 동일성
코틀린에서는 자바와는 다르게 객체를 비교할 때 비교 연산자를 사용하면 자동으로 `compareTo()` 를 호출한다.

자바에서 동일성은 `==` 연산자를 사용하여 비교한다. 하지만 코틀린에서는 `===` 연산자를 사용하여 객체의 동일성을 비교한다. 간접적으로 `==` 연산자를 실행한다.

자바에서 동등성은 `equals()` 메소드를 사용하여 비교한다. 하지만 코틀린에서는 `==` 연산자를 사용하여 객체의 동등성을 비교한다. 간접적으로 `equals()` 메소드를 호출한다.

# 논리 연산자
Java와 완전히 동일하며, 따라서 Lazy 연산을 수행한다.

# 코틀린에만 있는 특이한 연산자
- `in`, `!in` 
  - 컬렉션이나 범위에 포함되어 있다, 되어 있지 않다
- a..b
  - a부터 b까지의 범위 객체를 생성한다.
- a[i]
  - a에서 특정 Index i로 값을 가져온다.
- a[i]
  - a에서 특정 Index i로 값을 넣는다.
