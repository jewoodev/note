# data class
class에 data 키워드를 사용하여 생성한다. data class는 기본적으로 equals, hashCode, toString, copy, componentN, destructuring 등이 자동으로 생성된다.

# enum class
싱글톤으로 생성되며, 컴파일러가 컴파일 타임에 해당 클래스에 어떤 코드가 있는지 다 알고 있다는 특징이 있다. 이로 인해 when 표현식을 사용할 때 else를 사용하지 않아도 된다.

# sealed class, sealed interface
내부에서 상속으로 계층 구조를 만들 되, 외부에서는 상속하지 못하게 하는 클래스와 인터페이스다.

- **특징**:
  - 컴파일 타임 때 하위 클래스의 타입을 모두 기억한다.
    - => 런타임 때 클래스 타입이 추가될 수 없다.
  - 하위 클래스는 같은 패키지에 있어야 한다.
- Enum과 다른 점:
  - 클래스를 상속받을 수 있다.
  - 하위 클래스는 멀티 인스턴스가 가능하다.

이 또한 컴파일 타임 때 기억하는 특징 덕에 when 표현식을 사용할 때 else에 대한 처리가 필요하지 않다. 그리고 하위 구현체가 추가되거나 제거되었을 때 좀 더 쉽게 알아 차릴 수 있다.
